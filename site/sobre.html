Sobre a aplicação

A aplicação SearchImg, integra dois serviços do Amazon AWS, o Rekognition, famoso serviço de Visão Computacional, e o ElasticSearch, que funciona como um servidor de buscas distribuídos. Aplicamos o Rekognition para detectar objetos nas imagens que fizermos o upload pela aplicação. Logo após a detecção de objetos, a aplicação vai inserir o resultado em um documento no ElasticSearch, para que posteriormente possa ser buscado os dados que estiverem na plataforma. Além dos dois serviços, foi utilizado outros para facilitar a integração entre os serviços, como o Lambda, para provisionar a execução da aplicação, quando novas imagens forem upadas. Também utilizamos API Gateway para facilitar a integração com a busca no ElasticSearch, gerando uma API Rest que pode ser consumida por aplicações de terceiros.
